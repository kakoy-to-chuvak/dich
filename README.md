## <a id="description"></a>Описание
Простое приложение для создания, сохранения и редактирования маршрута робота.

#
## <a id="using"></a>Использование

#### Основные сочетания клавиш
| Сочетание / клавиша  | Действие           |
|--------------------- |--------------------|
| **Ctrl + O**         | Открыть файл       |
| **Ctrl + S**         | Сохранить файл     |
| **Ctrl + Shift + S** | Сохранить файл как |

#### Контексное меню
| Кнопка                 | Действие                                             |
|------------------------|------------------------------------------------------|
| **ПКМ**                | Открыть контекстное меню                             |
| **Add point**          | Создаёт точку маршрута и добавляет в конец текущего  |
| **Add point to start** | Создаёт точку маршрута и добавляет в начало текущего |
| **Delete point**       | Удаляет выбранную точку маршрута                     |


### Перкдвижение точек маршрута
| Сочетание / клавиша  | Действие                                                                                    |
|--------------------- |---------------------------------------------------------------------------------------------|
| **ЛКМ**              | Взятие и перемещение точки ( ниже представлены режимы перемещения точек )                   |
| **Shift**            | Режим перемещения точки по вертикали, горизонтали и диагонали                               |
| **Ctrl**             | Режим перемещения точки относительно линии между следующей и предыдущей                     |
| **Alt**              | Режим перемещения точки по вертикали, горизонтали и диагонали относительно предыдущей точки |
| **Alt + Shift**      | Режим перемещения точки по вертикали, горизонтали и диагонали относительно следующей точки  |

### Дополнительные функции
При запуске приложения из консоли, в качестве аргумента можно указать файл, который нужно открыть:
```bash
main.exe {file name}
```



## <a id="installing"></a>Установка

1. Скачайте репозиторий:
   ```bash
   git clone https://github.com/kakoy-to-chuvak/Robot-path-planner
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd Robot-path-planner
   ```
3. [Установите SDL3.](https://wiki.libsdl.org/SDL3/README-cmake)
   
4. [Установите SDL3_image](https://github.com/libsdl-org/SDL_image/blob/main/INSTALL.md)
   
5. [Установите SDL3_ttf](https://github.com/libsdl-org/SDL_ttf/blob/main/INSTALL.md)

6. Укажите в MakeFile параметры:
   - В переменой **SDL_INCLUDE_DIR** укажите папку с папками с ".h" файлами
     Ожидаемая структура **SDL_INCLUDE_DIR**:
     ├── SDL3
     │   ├── SDL.h
     │   ├── SDL_assert.h
     │   └── ∙ ∙ ∙
     ├── SDL3_image
     │   └── SDL3_image.h
     ├── SDL_ttf
     │   ├── SDL_textengine.h
     |   └── SDL_ttf.h
   - В переменой **SDL_LIBS_DIR** укажите папку с объектными файлами динамических библиотек
     Ожидаемая структура **SDL_LIBS_DIR**:
     ├── libSDL3.xxx
     ├── libSDL3_image.xxx
     └── libSDL3_ttf.xxx
     Где ".xxx" расширение объектного файла динамической библиотеки в вашей ОС

   - В переменной **SDL_LIBS_EXTENSION** укажите расширение объектного файла динамической библиотеки в вашей ОС

7. Скопируйте файлы динамических библиотек в **/build**

8. Запустите команду **make**:
   ```bash
   make
   ```
9. Готово! Теперь можно запускать **main** из **/build**
